Двухигольная машина челночного стежка для средне-тяжелых тканей с двойной системой продвижения материала, в который входит нижняя транспортирующая рейка, а также игольный двигатель ткани. Такой механизм позволяет получить отличное качество шитья и беспосадочную строчку на всех участках.

Механизм отключения игл позвояет Вам прокладывать двойную параллельную строчку даже на поворотах, по средствам отключения одной из игл.

Машина не заменима при выполнении отделочной строчки на клапанах карманов, манжетах и поясах, т.е. там, где необходим ровный красивый угол без перетяжки нитей.

Не устнавливая одну из игл, машину можно использовать как одноигольную прямострочную.

Особенности модели:
  • Предназначена для стачивания средне-тяжелых материалов двумя параллельными строчками
  • Машина имеет систему автоматической смазки.
  • Есть функция отключения иглы
